# Yuki Music 🎵

Glassmorphism music player with ad-free YouTube streaming.

## Download

**[⬇️ Download Latest Release](https://github.com/maq2008/YukiMusic/releases/latest)**

- Windows 10/11 (64-bit)
- Extract and run YukiMusic.exe
- No installation needed
- Ad-free streaming via YouTube

## Features

- Glass/frosted UI theme
- Search & stream any song
- Working seek + volume bars
- Next/Previous/Shuffle/Loop (off/all/one)
- Background async loading (no Not Responding)
- ~120MB RAM usage
